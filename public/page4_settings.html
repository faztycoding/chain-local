Pre-use Checklist
1. Start Server Run npm start and verify all 4 pages load correctly
2. Verify Latest Order Open Input and Output pages to verify the latest order matches
3. Test AI Connection Have YOLOv8 send data to /api/detect and check results on Output page
4. Check Reports Open Stats page to verify overview, daily, weekly, monthly, and history tabs
5. Backup Data Copy chain_data.db file for backup after each session
Usage Notes

The Start / Stop / Emergency Stop buttons are currently in simulation mode for demo and workflow testing only.

To connect real cameras or machines, use this web interface as a base and add hardware integration modules separately.

For deeper analysis, connect Power BI to the SQLite database or use the export API provided by the system.
