ตั้งค่าระบบ & บำรุงรักษา

หน้านี้ใช้สำหรับตรวจสอบสถานะระบบและข้อมูลที่จำเป็นสำหรับการดูแลระบบต้นแบบในหน้างาน

ข้อมูลระบบ
เซิร์ฟเวอร์หลัก Node.js + Express บนเครื่องภายในหน้างาน
ฐานข้อมูล SQLite ไฟล์ chain_data.db
พอร์ตใช้งาน http://localhost:3000
จุดเชื่อมต่อ AI POST /api/detect
การส่งออกข้อมูล GET /api/stats/export และ Views สำหรับ Power BI
รายการตรวจสอบก่อนใช้งาน
1. เปิดเซิร์ฟเวอร์ รัน npm start แล้วตรวจว่าหน้าเว็บเปิดได้ทั้ง 4 หน้า
2. ตรวจคำสั่งล่าสุด เปิดหน้า Input และหน้า Output เพื่อดูว่า order ล่าสุดขึ้นตรงกัน
3. ตรวจการเชื่อมต่อ AI ให้ฝั่ง YOLOv8 ยิงข้อมูลเข้า /api/detect แล้วดูผลที่หน้า Output
4. ตรวจรายงาน เปิดหน้า Stats เพื่อตรวจภาพรวม รายวัน รายสัปดาห์ รายเดือน และประวัติ
5. สำรองข้อมูล คัดลอกไฟล์ chain_data.db ออกเก็บตามรอบงาน
หมายเหตุการใช้งาน

ปุ่ม เริ่มทำงาน / หยุดทำงาน / หยุดฉุกเฉิน ยังเป็นโหมดจำลองสำหรับเดโมและทดสอบลำดับการทำงานเท่านั้น

หากต้องต่อกล้องหรือเครื่องจักรจริง ให้ใช้หน้าเว็บชุดนี้เป็นฐาน แล้วเพิ่มส่วนเชื่อมต่อฮาร์ดแวร์แยกภายหลัง

หากต้องการวิเคราะห์เชิงลึกเพิ่มเติม ให้เชื่อม Power BI เข้ากับ SQLite หรือ API ส่งออกที่ระบบเตรียมไว้
